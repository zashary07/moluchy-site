# Moluchy LLC

Authorized retail distribution partner — corporate website.

Built with vanilla HTML, CSS, and JavaScript. Deployed via GitHub Pages.

**Live:** [moluchy.com](https://moluchy.com)

## Stack

- Static HTML/CSS/JS (no framework)
- Google Fonts: Cormorant Garamond + Inter
- GitHub Pages hosting
- Custom domain via CNAME

## Brand

- **Primary color:** `#500724` (Wine)
- **Background:** `#F8F2EC` (Cream)
- **Typography:** Cormorant Garamond (display) + Inter (body)
- **Logo:** Baskerville Italic wordmark + dotted divider + LLC
